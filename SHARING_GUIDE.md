## 👤 三、用户前置条件

### 3.1 系统环境要求

**硬件要求：**
- 💻 电脑（推荐）或手机
- 🎤 麦克风（用于语音识别）

**软件要求：**
- 🌐 浏览器（必需）
  - ✅ Chrome 33+（强烈推荐）
  - ✅ Edge 79+
  - ⚠️ Safari 14.1+（部分功能）
  - ❌ Firefox（不支持语音识别）

**网络要求：**
- 🌐 首次使用需要网络（加载PDF.js和Tesseract.js）
- 🔌 后续可离线使用基础功能
- ☁️ 使用AI总结需要持续网络连接

### 3.2 账号与权限

**不需要账号注册！**
- ✅ 无需登录
- ✅ 无需注册
- ✅ 直接使用

**浏览器权限：**
- 🎤 需要麦克风权限（首次使用时浏览器会询问）
- 💾 本地存储权限（保存API配置）

### 3.3 首次使用检查清单

用户首次使用前，请确认：
- [ ] 使用 Chrome 或 Edge 浏览器
- [ ] 麦克风设备正常工作
- [ ] 网络连接正常（首次使用）
- [ ] 已阅读 [README.md](./README.md)

## 🔒 七、安全与稳定性保障

### 7.1 数据安全

**向用户说明：**
- ✅ 所有数据保存在浏览器本地
- ✅ 录音不上传服务器
- ✅ API密钥仅保存在本地
- ✅ 简历仅在当前会话内存中

**安全建议：**
- ⚠️ 不要在公共电脑使用
- ⚠️ 定期清除浏览器数据
- ⚠️ 不要分享包含API密钥的浏览器配置

### 7.2 稳定性保障

**技术保障：**
- 纯前端应用，无服务器依赖
- 使用CDN加载外部库，稳定可靠
- 提供本地总结作为AI的备选方案

**降级策略：**
- AI总结失败 → 自动使用本地总结
- 语音识别失败 → 提示用户检查
- 文件解析失败 → 提供错误信息和重试
