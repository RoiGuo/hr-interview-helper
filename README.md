# HR面试助手 - 首次电话沟通

## 🚀 快速开始

### 方式一：直接打开（最简单）

1. 下载应用文件（index.html、style.css、app.js）到同一文件夹
2. 双击 `index.html` 在浏览器中打开
3. 开始使用！

### 方式二：本地服务器（推荐）

使用 Python 启动本地服务器：

```bash
# Python 3
python3 -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器访问：`http://localhost:8000`

## ✨ 功能特性

### 🎙️ 实时语音记录
- 基于浏览器Web Speech API的实时语音转文字
- 支持中文语音识别
- 实时显示录音状态和计时器

### 📄 简历上传与分析
- **支持格式**：PDF、TXT、JPG、PNG、WebP
- **拖拽上传**：支持直接拖拽文件到上传区域
- **自动解析**：使用PDF.js解析PDF，Tesseract.js识别图片文字
- **智能分析**：自动提取候选人信息、工作经历、技能
- **HR建议**：基于简历提供首次沟通建议和话术

### 📞 首次电话沟通追踪
实时追踪6个关键维度：
- 💼 当前工作内容
- 🛠️ 专业技能
- 💬 沟通能力
- 🧩 问题解决能力
- 💰 薪资期望
- 🚪 离职原因

### 📋 自动总结
- 每3秒自动更新通话总结
- 支持自定义AI API配置
- 提供本地总结作为备选方案

### 💬 延伸聊点建议
- 基于简历内容和通话进度
- 动态推荐可深入讨论的话题
- 确保首次沟通全面覆盖

### 📤 报告导出
- 一键导出完整的面试报告
- 包含候选人信息、沟通状态、通话实录、总结

## 📖 使用说明

### 系统要求

**浏览器要求：**
- Chrome 33+（推荐）
- Edge 79+
- Safari 14.1+
- Firefox 不支持语音识别功能

**网络要求：**
- 需要网络连接以加载PDF.js和Tesseract.js库
- 如需使用AI总结功能，需要连接配置的API服务

**硬件要求：**
- 麦克风（用于语音识别）

### 首次使用步骤

#### 1. 打开应用
直接在浏览器中打开 `index.html` 文件。

#### 2. （可选）配置API
点击右上角「⚙️ API配置」按钮，配置AI服务：
- 选择API提供商
- 填写API端点
- 输入API密钥
- 设置模型名称
- 点击「保存配置」

#### 3. 上传简历
- 点击「选择文件」或直接拖拽简历文件到上传区域
- 等待解析完成
- 查看候选人信息和HR建议

#### 4. 开始录音
- 点击「▶️ 开始录音」
- 允许浏览器访问麦克风
- 开始与候选人通话

#### 5. 使用沟通追踪
通话过程中，系统会自动：
- 实时显示语音转录
- 每3秒更新总结
- 追踪沟通状态
- 推荐延伸聊点

#### 6. 结束录音
- 点击「⏹️ 停止录音」
- 查看完整总结
- 点击「📄 导出报告」保存记录

### 前端技术栈

## ❓ 常见问题

### Q: 为什么语音识别不工作？
A: 
- 确保使用Chrome或Edge浏览器
- 检查是否允许浏览器访问麦克风
- 确认麦克风设备正常工作

### Q: 支持哪些语言的语音识别？
A: 当前默认配置为中文（zh-CN），如需修改可在app.js中调整recognition.lang。

### Q: 数据安全吗？
A: 
- 所有数据保存在浏览器本地
- 语音识别在浏览器端完成
- 仅在使用AI总结时会向配置的API发送数据

### Q: 可以离线使用吗？
A: 
- 基础录音功能可以离线使用
- 首次加载需要网络以下载PDF.js和Tesseract.js
- AI总结功能需要网络连接

更多问题请查看 [FAQ.md](./FAQ.md)。
